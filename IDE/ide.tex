\section{K-map Implementation}
Using the boolean logic output F can be expressed in terms of the inputs X,Y,Z with the help of the following Kmap.
\\
\\
\\
	 \begin{center}
     \begin{karnaugh-map}[4][2][1][$YZ$][$X$]
        \maxterms{0,1,5}
        \minterms{2,3,4}
        %\minterms{7,6}
        \terms{6}{$X$}
		\terms{7}{$X$}   
        \implicantedge{4}{4}{6}{6}
        \implicant{3}{6}
       
    \end{karnaugh-map}
